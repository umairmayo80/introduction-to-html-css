Semantic Elements in HTML

<header>

The <header> element represents the header section of a document or a section. It usually contains a logo, navigation menu, or heading.

Example: <header>Welcome to Our Website</header>

<nav>

The <nav> element represents a section of the page that contains navigation links.

Example: <nav><a href="/home">Home</a> <a href="/about">About</a></nav>

<main>

The <main> element represents the main content of the document. It should not contain headers, footers, or sidebars.

Example: <main><p>This is the main content of the page.</p></main>

<article>

The <article> element represents a self-contained, independent piece of content that could be distributed and reused independently.

Example: <article><h2>Article Title</h2><p>This is the content of the article.</p></article>

<section>

The <section> element represents a thematic grouping of content. It can be used to divide the content into different sections.

Example: <section><h2>Section Title</h2><p>This is the content of the section.</p></section>

<aside>

The <aside> element represents content that is tangentially related to the content around it. It is often used for sidebars or related content.

Example: <aside><p>This is the sidebar content.</p></aside>
