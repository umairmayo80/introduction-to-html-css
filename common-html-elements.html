Common HTML Elements

<h1> to <h6>

Heading elements are used to define headings and subheadings. They represent a hierarchical structure for your content.

Example: <h1>Main Heading</h1>

<p>

The <p> element is used for paragraphs of text. It represents a block of text that forms a cohesive unit.

Example: <p>This is a paragraph.</p>

<img>

The <img> element is used to display images on the webpage.

Example: <img src="example.jpg" alt="Example Image">

<a>

The <a> (anchor) element is used for creating links to other web pages or resources.

Example: <a href="https://www.example.com">Visit Example Website</a>

<ul> and <li>

The <ul> element is used for creating unordered lists, and the <li> element is used for list items within the list.

Example: <ul>
  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>
</ul>

<table>, <tr>, <th>, and <td>

These elements are used to create tables to organize tabular data.

Example: <table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

<div>

The <div> element is used as a container to group other elements together or to apply styles to a section of the page.

Example: <div style="background-color: lightblue; padding: 10px;">
  <p>This is a <div> element used as a container.</p>
</div>

Divs are commonly used in conjunction with CSS for layout purposes and to apply styles to groups of elements.
